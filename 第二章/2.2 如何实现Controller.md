# 如何实现Controller

## AngularJS中的Controller

``` html
<html ng-app>
    <head>
        <meta charset="utf-8">
        <script src="https://code.angularjs.org/1.2.9/angular.js"></script>
    </head>
    <body>
        <div ng-controller="helloAngular">
            <p>{{greeting.text}},Angular</p>
        </div>
        <script src="HelloAngular_MVC.js"></script>
    </body>
</html>
```

``` javascript
// HelloAngular_MVC.js
function helloAngular($scope) {
    $scope.greeting = {
        text: 'Hello'
    }
}
```

Controller实现方式

``` flow
st=>start: 数据模型
e=>end: 视图1
e=>end: 视图2
```
